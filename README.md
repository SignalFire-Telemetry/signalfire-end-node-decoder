# signalfire-end-node-decoder
Payload decoders for SignalFire End Nodes

### Supported Network Servers:
* ChirpStack V3
* ChirpStack V4
* The Things Network (TTN)

### Decoded Variables:
#### FPort 1
* pressure_psi
* pressure_avg_psi
* pressure_max_psi
* pressure_min_psi
* status
* low_alarm_active
* high_alarm_active
* low_battery
* sensor_error

#### FPort 2
* battery_v
* version
* pressure_span_psi

#### FPort 3
* low_alarm_threshold
* high_alarm_threshold
* alarms_enabled
* low_alarm_enabled
* high_alarm_enabled
* report_interval_sec
* sample_interval_sec

#### Decode Errors
* error_msg
